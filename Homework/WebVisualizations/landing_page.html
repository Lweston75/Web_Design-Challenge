Summary: Latitude vs. X

The purpose of this project was to analysis how weather changes as you get closer to the equator. To accomplish this analysis, we fist pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we ysed Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at inclueded: temperature, cloudiness, wind speed, and humidity. This site provides the source data and the visualizations creaded as part of the analysis, as well as explanations and descriptions of an trends and correlations witnessed.
